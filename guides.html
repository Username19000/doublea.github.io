Getting Started

First Steps on the Server

  1. Read the /rules command
  2. Use /spawn to return to the main hub
  3. Explore and find a place to build
  4. Start gathering resources and building your base

Land Claims

Protecting Your Builds

  1. Get a golden shovel by using /kit tools
  2. Right-click one corner of your claim area
  3. Right-click the opposite corner to complete the claim
  4. Use /trust [player] to allow friends to build
  5. Use /untrust [player] to remove access
üí° Tip: You earn more claim blocks the longer you play! Use /claimlist to view all your claims.

Chest Locking

Securing Your Items

  1. Place a sign on your chest
  2. Type [Private] on the first line
  3. Your name will auto-fill on the second line
  4. Add trusted players on lines 3 and 4
üí° Tip: Works on chests, furnaces, doors, and more!

Player Shops

Setting Up a Shop

  1. Place a chest with items you want to sell
  2. Place a sign on the chest
  3. Line 1: Leave blank
  4. Line 2: Quantity
  5. Line 3: B [price] : S [price]
  6. Line 4: Item name or leave blank
Example:
Line 2: 64
Line 3: B 10:S 5
(Buy 64 for $10, Sell 64 for $5)

Useful Commands

/spawn Teleport to spawn
/sethome [name] Set a home location
/home [name] Teleport to your home
/tpa [player] Request to teleport to a player
/balance Check your money
/pay [player] [amount] Send money to another player
